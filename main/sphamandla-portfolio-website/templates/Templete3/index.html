How to create a professional website

Author:Tella Smith,13 May 2018

Introduction

Every professional business should have a professional website. There are lot of tools out there to create professional website but without a professional web programmer you are liable to have a bad website if you do not know the basic technique that are considered when creating a website.Today I will show you 3 tips to build a professional website for you busness or personal branding.

  • Step 1

    Responsive Design for all screen devices

    Every website should have a responsive layout when viewed in any device. Alot of times people use their smart phones to search web content,so it is important that your website is designed for smartphones as well as desktop computers.There are frameworks such as bootstrap,angular material for web programmers, can also use CSS media queries and CSS grids for responsive layout.Check your templete on your CMS website build if it is responsive by simply resizing it or use some emulators for testing.

  • Step 2

    Browser Compability for all browser support.

    Making sure your website is looking and behaving the same in all browser or atleast morden browser is the requirement because you do not know which browser app user will choose to view your website.To resolve this as a web designer,use web prefixes in your CSS code such as -webkit- for chrome and safari,and -moz- for firefox, and -ms- for microsoft IE.You can use grunt service on npm install and also some online tools to automatically add prefixes for browser support.For none developers checkbok your templete by viewing them on different browsers and use some plugins on your CMS.

  • Step 3

    Accessibility for disabled people.

    Using chrome vox as a testing tool is essential to ensure a great user expirience for everyone entering your website.Use html elements as the way they are intended to be used is highly recommended to avoid mistakes of ignoring disabled people using your website. Some html elements are used such as image alt and aria could helpfpr developers.
